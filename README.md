# StarDB - 崩坏：星穹铁道攻略站

## 项目概述
这是一个基于 stardb.info 域名的崩坏：星穹铁道(Honkai: Star Rail)游戏攻略网站。

## 网站特色
- 📊 **角色节奏榜** - 定期更新的角色强度排名
- ⚔️ **忘却之庭攻略** - 混沌回忆、纯虚构、末日幻影满星阵容推荐
- 🌌 **模拟宇宙指南** - 各难度通关攻略与命途选择策略
- 👤 **角色培养攻略** - 光锥推荐、遗器搭配、配队建议
- 💎 **素材资源指南** - 星琼获取、材料 farming 路线

## SEO 策略
1. 定期更新"角色节奏榜"和"忘却之庭阵容推荐"

## 技术栈
- HTML5
- CSS3 (自定义样式，响应式设计)
- Vanilla JavaScript
- 静态网站架构

## 本地运行
直接在浏览器中打开 `index.html` 文件即可预览。

或使用任意 HTTP 服务器：
```bash
# Python 3
python -m http.server 8000

# Node.js (需要安装)
npx serve
```

## 部署
将项目文件上传至支持静态网站的托管服务，配置域名 stardb.info 即可。
